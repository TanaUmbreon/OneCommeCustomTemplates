=============================
わんコメ カスタムテンプレート
(OneComme Custom Templates)
=============================

配信者向けのコメントアプリ「わんコメ」を使ったカスタマイズ済みのテンプレートです。
ゲーム「Library Of Ruina」の幻想体戦などで使われているメッセージ風にコメントを表示させることができるようになります。

-------------------------
2. 利用するための前提条件
-------------------------

【必須】
・コメントアプリ「わんコメ」がPCにインストールされていること。
【任意】
・フォント「ロゴたいぷゴシック」がPCにインストールされていること。
　※「Library Of Ruina」で使われているフォントに似せるために使用しますが、インストールしていなくても問題ありません。
　　この場合、OSの標準フォントを使用します。

これらアプリやフォントのダウンロードおよびインストールの手順は割愛します。
詳細は各リンク先ページで手順を確認してください。

【わんコメ】
https://onecomme.com/

【ロゴたいぷゴシック】
http://www.fontna.com/blog/1226/


-------------------------------------
3. インストール手順・アップデート手順
-------------------------------------

1. このファイルと同じ場所にある「templates」フォルダーを開きます。
2. 「lor-like-comment」フォルダーをコピーします。
3. 「わんコメ」を起動し、画面右上の「…」ボタンから「フォルダを開く」→「テンプレートフォルダ」メニューをクリックします。
4. 表示された「エクスプローラー」の場所に[手順2]のフォルダーをペーストします。
5. 「わんコメ」の画面右上の「…」ボタンから「テンプレート」メニューをクリックします。
6. 表示されたテンプレートの一覧に「lor-like-comment」が出てくるようになればインストール完了です。

これにより、他のテンプレートと同様の操作で利用することができるようになります。

このテンプレートをアップデートする場合も手順は同様です。
ただし、[手順4]のペーストを行った時にファイルを全て上書きするようにしてください。


-----------------------
4. アンインストール手順
-----------------------

1. 「わんコメ」を起動し、画面右上の「…」ボタンから「フォルダを開く」→「テンプレートフォルダ」メニューをクリックします。
2. 表示された「エクスプローラー」の場所にある「lor-like-comment」フォルダーを削除することでアンインストール完了です。
